Projects

Election System Enabled by Blockchain

Created a Blockchain based Secure Voting System. This is meant to tackle the tampering of Voting Machine tampering in India. It does so by utilizing two factor authentication of each vote by QR code scanning and generation of random key value pairs for each voter on both the client and server side. It allows for vote verification by the voter after the election is over. It is modelled around the Estonian model of blockchain voting.

Intellectual Property Management System using Blockchain

Designed an Intellectual Property Management system which used a smart contract written in Solidity at its core. This way, the issued Intellectual Property object would have inherent trackability, thus authenticating it using a universal source of truth. It would also allow for expedited resolving any ownership conflicts and any legal issues that may arise from them. This project is being done under the advisory of Professor Rajorshi Sen Gupta at BITS Pilani Goa Campus.
